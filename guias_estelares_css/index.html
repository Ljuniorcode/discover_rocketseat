Aprendendo CSS

  • Global: *
  • Element / Type selector `h1, h2, p, div`
  • ID Selector `#box, #container`
  • Class Selector `.red, .m-4`
  • Attribute selector, Pseudo-class, Pseudo-element e outros

Box Model

  • margin: espaço da caixa do lado de fora
  • pading: espaço interno da caixa
  • border: borda da caixa: ex: 2px solid red

Caixas

Quase tudo são caixas, posicionamentos, tamanhos,
espaçamentos, bordas, cores, podem ficar uma ao lado da outra, ou acima, elementos HTML são caixas

Importando fontes no css

Criar o arquivo styles.css separado do html

@import url('https://fonts.googleapi....')

Cascata

É Lido de cima pra baixo e levado em conta 3 fatores

  1. Origem do estilo
  2. Especificidade
  3. Importância

Origem do estilo

inline > tag style > tag link

Cascata

a leitura das configs vem de cima pra baixo

  • 1º: style dentro da tag < h1 style='color:red; > é a mais forte
  • 2º: dentro da tag <styles>
  • 3º: arquivo externo styles.css

Especificidade

  • classes: intermediário
  • id: mais forte
  • tag <qualquer> é o mais fraco

regra !important

sobrescreve tudo que você colocou nas outras configs

Outras regras

  • @import - inclui um css externo
  • @media - regras condicionais para dispositivos
  • @font-face - fontes externas
  • @keyframes - animações

  • -moz-background-clip: text (mozilla firefox)
